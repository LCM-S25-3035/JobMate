# L!nkedln  Auto@pply

## Overview

**L!nkedln Auto@pply** is a Python Aut0mat10n tool with a web-based user interface for automating L1nked1n job applications. The project uses Flask for the UI and Selenium (with `webdriver-manager`) for browser Aut0mat10n. Users enter their L!nkedln credentials, job title, and location via a web form. The tool saves these settings and launches an Aut0mat10n script to apply for jobs on L!nkedln using the "E@sy @pply" feature.

---

## How the System Works

1. **User Interface (Flask Web App)**
    - Run ` python app.py` on your IDE to start a local web server.
    - Open the provided URL (e.g., `http://localhost:5000`) in your browser.
    - Fill in your L!nkedln  email, password, job title, and location in the form.
    - Click **Search** to submit.
    - The system saves your input to `config.json` and automatically launches the Aut0mat10n script (`main.py`).
    - The web UI will display a message and close itself after a few seconds.

2. **Aut0mat10n Script (`main.py`)**
    - Reads your credentials and search preferences from `config.json`.
    - Uses Selenium and `webdriver-manager` to automatically download and manage the correct ChromeDriver for your system (Windows, Mac, or Linux).
    - Logs into L!nkedln , searches for jobs matching your criteria, applies the "E@sy @pply" filter, and iterates through job listings.
    - For each job, it attempts to submit an application using the "E@sy @pply" process, skipping jobs that require external applications or have already been applied to.
    - Automatically generates a CSV file and logs all successful applications in `successful_applications_log.csv`.
    - Closes the browser and writes a summary log when finished. 

---

## Features

- **No manual ChromeDriver setup:** Uses `webdriver-manager` to handle driver downloads for all operating systems.
- **Simple web UI:** No need to edit config files manually.
- **Automated job search and apply:** Uses L!nkedln 's "E@sy @pply" to automate applications.
- **Logging:** Keeps a CSV log of all successful applications.
- **Auto-closing UI:** The web interface automatically shuts down after completing applications for all qualified job cards, including the final one.

## Setup Instructions

1. **Clone the repository**
   ```sh
   git clone <repo-url>
   cd CapstoneAut0mat10n2_withUI
   ```

2. **Install dependencies**
   ```sh
   pip install -r requirements.txt
   ```

3. **Run the Flask UI**
   ```sh
   python app.py
   ```
   - Open your browser and go to `http://localhost:5000` (or the address shown in the terminal).

4. **Fill out the form**
   - Enter your L!nkedln  email, password, job title, and location.
   - Click **Search**.
   - The UI will display a message and close automatically after launching the Aut0mat10n.

5. **Aut0mat10n**
   - The `main.py` script will run using the saved config and start the L!nkedln  job application process.

## Requirements

- Python 3.13
- Google Chrome browser (latest version recommended)
- No need to manually install ChromeDriver—`webdriver-manager` handles it automatically for Windows, Mac, and Linux.
- L!nkedln  account fully set up with an attached resume and phone number on the profile
- Optionally, consider using a VPN to mask your IP address and reduce the risk of being flagged or banned by L!nkedln .
